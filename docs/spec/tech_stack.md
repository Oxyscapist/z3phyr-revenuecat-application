# Tech Stack

## Frontend
- Recommended: Next.js or React SPA
- UI library: shadcn/ui or Chakra UI

## Backend
- Recommended: FastAPI or Node.js (NestJS/Express)
- Auth: JWT/session with server-side authorization checks

## Data
- Primary DB: Postgres
- Cache: Redis

## Infra
- Hosting: managed platform (Vercel/Render/Fly/Cloud Run)
- CI/CD: GitHub Actions

## Cost choices
- Prefer managed services and free tiers initially
- Enforce quotas/rate limits for expensive features
