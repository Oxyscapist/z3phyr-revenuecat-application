# UI/UX

## Core screens
- Landing
- Authentication/onboarding
- Work Item list view
- Work Item detail view
- Settings/profile

## UX principles
- Clear primary CTA on every core screen
- Minimize required steps in the main flow
- Fast, explicit feedback for success/failure states

## Accessibility
- Keyboard navigation support
- Visible focus states and contrast compliance
- Semantic labels for all interactive controls
