# User Stories

## Core flows
- As a target user, I want to complete the main work-item workflow, so that I can get value quickly.
- As a target user, I want to see my past work-items, so that I can track progress over time.
- As a target user, I want reliable feedback on failed actions, so that I can recover quickly.

## Admin/ops flows
- As an admin, I want visibility into system health and usage, so that I can maintain reliability and costs.

## Edge cases
- Unauthenticated user attempts a protected action
- Invalid payload shape or missing required fields
- Third-party integration timeout during write operation
